Rewrote `src/index.ts` to register three languages (`gitignore`, `gitconfig`, `gitattributes`) with `IEditorLanguageRegistry` and four file types in the document registry. Added `@codemirror/language`, `@jupyterlab/codemirror`, `@jupyterlab/ui-components` to `package.json` dependencies. Wrote 19 unit tests in `src/__tests__/jupyterlab_git_syntax_extension.spec.ts` using a MockStringStream class covering all three parsers.
